# 本文将讲述`setting.json`如何配置
> 为什么要有`setting.json`?  
> `setting.json`存储了所有的全局设置,没有它程序将不会正常运行
## 示例
```json
{
  "language": "en-us",
  "commandConfig": "command.json",
  "commandDir": "command",
  "debug": false
}
```

- language为全局语言,比如en-us,表示所有命令统一用英语输出/记录日志
- commandConfig为命令配置,command.json就表示所有命令配置都存储在command.json文件中
- commandDir为命令调用文件夹,在读取到了commandConfig之后调用文件会在这个文件夹里面调用
- debug,就是一个全局的开关,命令可以通过debug来修改一些逻辑
